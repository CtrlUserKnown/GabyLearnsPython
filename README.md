### ★ THE PRACTICE CABINET! (Rustlings-style Exercises) ★

Oh, oh, OH! This is my FAVORITE part! A self-checking practice system inspired by Rustlings! Each `Practice/Mod{N}.py` file looks like a REAL project module with `# TODO` markers and `raise NotImplementedError()` stubs. The Gaby replaces them with code and — WHOOSH — the checker says if it's CORRECT!

AND NOW, FOR OUR NEXT CHALLENGE — **THE PYLINGS TUI**!

That's right! I've built a FABULOUS keyboard-controlled Terminal User Interface! It's like a lazygit for learning Python! Just run:

```bash
pylings
```

And the Gaby will see a beautiful list of all exercises. Navigate with arrow keys, press ENTER to run a test, press `h` for hints, press `e` to edit the file, press `r` to refresh — it's a CONTROL PANEL for learning! Isn't that EXCITING? HAHAHA!

Not installed yet? Well then, let me wave my digital wand:

```bash
# Option A: The Full Install (builds + installs)
sh install.sh

# Option B: Quick play from the repo
./pylings-tui
```

The first time `install.sh` runs, it ALSO initializes the submodule that contains all the Practice exercises. It's like a PRIZE DISPENSER — POP! Everything the Gaby needs comes out of one tiny script!

If the Gaby prefers the classic command-line experience (no fancy TUI):

```bash
pylings next       # Run the next incomplete exercise
pylings list       # Show progress on all exercises
pylings 3          # Run a specific module
```

When ALL checks pass, remove the `# I AM NOT DONE` line from the file to mark it COMPLETE! Like hearing DING DING DING on a slot machine!

## How to play — THE GABY'S GUIDE!

1. **Start in the Training Room!** Open the `Lesson/` folder and go through the files in order. Read the comments — they're like the GAME MANUAL!

2. **Try the Bonus Round!** Once a lesson clicks, head to the `Assignments/` folder and show off what the Gaby has learned!

3. **Run the pylings!** Use the `pylings` command to practice with auto-graded exercises. The TUI makes it feel like a VIDEO GAME!

4. **Run the code!** Open lesson notebooks in VS Code or Jupyter. For assignments, run them in the terminal:
   ```bash
   python3 Assignments/assignment1.py
   ```
