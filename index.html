★ ABOUT THE ARCADE ★

What in the digital realm IS this magnificent contraption?

🏠

The Training Room

20 interactive Jupyter notebooks that teach Python from the ground up. Variables, loops, classes, pandas — THE WHOLE SPECTACULAR SHOW!

🎰

The Practice Cabinet

16 Rustlings-style modules with # TODO markers and NotImplementedError stubs. Fill in the blanks and let the checker say if it's CORRECT!

🏆

The Bonus Round

10 real-world assignments themed around a bakery. Each one is a BOSS LEVEL that combines everything learned so far!

🕹️

The Controller Console

The Pylings TUI — a keyboard-navigable terminal interface written in Swift. Like lazygit, but for LEARNING! Navigate, run, get hints, edit — all without touching the mouse!

🎯

The Mission

To take the Gaby — our wonderful contestant — from absolute beginner to confident Python programmer through an engaging, game-like curriculum. One lesson, one exercise, one high score at a time.

◆ THE CONTROLLER CONSOLE ◆

The Pylings TUI — a MAGNIFICENT keyboard-driven terminal interface!

Oh, oh, OH! This is my FAVORITE part! The Pylings TUI is a Rustlings-style exercise runner with a BEAUTIFUL keyboard-driven interface. It's like a lazygit for LEARNING!

  • ⭐ Navigate with ↑↓ or jk
  • ▶ Press Enter to run a test
  • 💡 Press h for hints
  • ✏️ Press e to edit the file
  • 🔄 Press r to refresh
  • ❌ Press q to quit

Built in PURE SWIFT with ZERO dependencies. Universal binary for arm64 + x86_64. It's a CONTROL PANEL for learning! Isn't that EXCITING? HAHAHA!

# Install the TUI
sh install.sh

# Or run directly
./pylings-tui
